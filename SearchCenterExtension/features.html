Previous Features & updates:

Manage Engines
You can Drag engines into the trash to delete them
SearchBar
Fixed page reloading & double pages bugs (thanks for the info luke)
Popup
tab search can now be used from the popup
tab search
If you prefix a default search with ?, search center will scan your open tabs and bring that tab to the front, e.g: ?gmail
Search bar
  • middle clicking an engine no longer brings up the scroll cursor
  • right click an engine no longer brings up the context menu
  • will display correctly on more websites
  • can disable search bar in options
  • engines have hover titles
Search Bar

Ctrl + q will bring up a search bar. You will need to reload any existing pages to see this work.

Autoseach (enter), left and middle click are implemented, I'll be adding features and bringing more of the pop up features over the next several releases.

The search bar doesn't render correctly on many sites, but it's at a point where it's worth releasing

Donate

You can now donate from the options page!

If you choose to donate some cash (highly recommended), I will probably definately buy a nice cold beer or two :)

Pin engines
right click will now pin an engine, left or middle click with run the search on all the pinned engines & the selected engine
Small Icons
enable small engines in options
Suggestions
  • Suggestions shouldn't break or slow the UI with a bad network connection
  • Suggestions can disabled in options
History
Right clicking the search box will bring up this sessions search history
Export
Copy or save your search engines by exporting them from the options page
Suggestions
Typing now brings up suggestions
Follow on Twitter!
Follow my projects on twitter, give suggestions and be notified of updates and new projects
Search history
Use the up & down arrows and mouse wheel to scroll through previous searches. No longer remembers the last search between chrome sessions (complete close of all browser windows)
Set default search to previous site
In options, you can set the default engine to be the last one you used
Edit Engines
You can edit an websites details by going to the options page and clicking the edit icon
Custom default search
The default search will be the first website in the list, change the order in the options
Right click multi-search
Right clicking an engine will not close the popup; so you can search multiple engines at once
Shortcut keys
Typing / followed the the first letter of the domain will open the search page for the engine eg "food /w" will search for "food" in wikipedia. You can add more switch to do a multisearch eg /w /i will search wikipedia & imdb.
New Version notifcation
Search Center popup will nofity you when a new version is installed, so you can learn & start using the enhancements straight away.
Reorder website list
from the options page you can change the order the the websites are displayed but click the arrow buttons.
Directly open a url
If you select and url and do a default search (enter or clicking the icon to the left of the text box). Instead of running a search, search center will navigate directly the url.
Search for selected text
When you select text on the page Search Center will automatically fill its input with the selected text
Remember address bar searches
If you do a google search from the address bar, search center will remember that search. If you do a tabbed search from the address bar Search Center will remember the search if it is in it's list of engines,
Middle Click
If you click on a search engine with the middle mouse button it will be opened in a new tab. This behaviour is customisable in the options page.
Open Search
Open Search data is sometimes included in a page that has a search box. If you go to one of these sites and click the Add Current Site the search engine will be added.
Site Learning
If you use searchcenter as your search term at a search site and then click the Add Current Site (after doing the search), Search Center will see this and add the search engine to its list.
Manual Add
Manually add websites from the options menu, by entering the name, icon location and search Url.
MyCroft Support
Go to mycroft and do a search or click on the Top 100. After the page has loaded enough any search engines that support OpenSearch will have a Search Center icon next to them. Clicking on this icon will add that search engine to Search Center.
Current Site Search
Pressing Shift+Enter in the Search Center popup will cause Search Center to do a google search of the site (in the selected tab) for the terms in the input box.
Icon Cacheing
Icons displayed in Search Center are cached on the users computer so that Search Center doesnt have to get them everytime Chrome is started. This make the extension alot more responsive, especially for dialup users.
